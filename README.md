# Kebutuhan project
1. php 8.1
2. composer
3. web server apache
4. mysql

# Cara menjalankan project
1. clone project di laptop/server
   - `git clone https://github.com/nandakharismas22/tracer-study/edit/main/README.md`
3. hapus file bawwaan
   - `del composer.lock`
   - `del vendor`
4. copy enviroment
   - `cp .env.example .env`
5. installasi vendor
   - `composer install`
6. Jalankan project
   - `php artisan serv`
